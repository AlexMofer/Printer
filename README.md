Printer
=========

介绍
---

标准ESC-POS命令打印，固定IP或蓝牙打印，支持黑白图片打印

使用
---

**引用:**
```
dependencies {
    ...
    implementation 'io.github.alexmofer.printer:printer:2.1.2'
    ...
}
```
